# Analisis-Ranking-Kemiskinan-Jawa-Timur-2020
Project ini membahas analisis ranking tingkat kemiskinan kabupaten/kota di Jawa Timur tahun 2020 menggunakan data dari Badan Pusat Statistik (BPS). Analisis dilakukan untuk mengidentifikasi daerah dengan tingkat kemiskinan tertinggi dan terendah serta melihat distribusi kemiskinan antar wilayah di Jawa Timur.
